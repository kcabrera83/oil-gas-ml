# ML Oil & Gas Petrochemical

Repositorio de proyectos de ciencia de datos aplicados a la industria de petróleo y gas utilizando Machine Learning, Inteligencia Artificial y Python.

El objetivo es ofrecer soluciones de código abierto para el análisis de datos de producción, predicción de yacimientos y optimización de procesos mediante modelos predictivos.

## 🚀 Características Principales

- **Pipeline Automatizado:** Ejecuta flujo completo de ETL, entrenamiento y evaluación con un solo comando.
- **Generación de Reportes:** Crea automáticamente visualizaciones de correlaciones, importancia de características y desempeño del modelo.
- **Modularidad:** Código organizado en módulos reutilizables (`src/`) siguiendo buenas prácticas de ingeniería de software.
